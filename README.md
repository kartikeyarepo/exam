**The Exam Portal: A Complete Online Testing Solution**

The Exam Portal is a full-stack application designed to digitize the entire examination lifecycle for educational institutions. By providing dedicated portals for teachers, students, and administrators, it simplifies everything from test creation to result analysis.

**Key Functionalities:**

- **Secure Access:** The application uses JWT tokens to manage secure, role-based access, ensuring that teachers and students only see the features relevant to them.
- **Intuitive Quiz Builder:** Teachers can easily build exams using a variety of question formats, with full support for creating and managing multiple-choice questions (MCQs).
- **Real-Time Results:** The system automatically grades quizzes upon submission, providing students with immediate performance reports and teachers with valuable data.
- **Timed & Auto-Submitted Exams:** To maintain integrity, each quiz includes a visible timer. If a student exceeds the time limit, the test is automatically submitted with the answers recorded up to that point.
